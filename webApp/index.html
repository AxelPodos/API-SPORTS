Frontend

  • Html5.
  • Css Estilos: Utilizados para algunas Tags, Estructuras del HTML, y la Barra de navegación.
  • Boostrap: Para la creación dinámica de tablas en pestañas Paises/Ligas y Jugadores.

Backend

  • Javascript.
  • Libreria Axios: Para consumir servicios de la API (AllSportsApi).
  • Libreria Jquery: Para el "bindeo" de algunos elementos del HTML a JS.

1° API

  • Llamado a /countries: "apiv2.allsportsapi.com/football?met=Countries".
  • Llamado a /leagues: "apiv2.allsportsapi.com/football?met=Leagues"
  • Interacción: Filtro que al buscar por el nombre del País, muestra ligas y torneos correspondientes al mismo.

2° API

  • Llamado a /topscores: "apiv2.allsportsapi.com/football?met=Topscorers".
  • Interacción: Filtro que al buscar por el nombre del Goleador, muestra propiedades como goles, asistencias, penales.
